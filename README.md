# 3d

## input.py
В интерактивном режиме позволяет вводить количество фигур, их вершины и грани.
На выход получаем фигуру, описанную в ply-файле.
![image](https://user-images.githubusercontent.com/37455205/114508727-5fda1d00-9c3d-11eb-8b87-9657971faec2.png)

## 3d.py
Программа строит такие объекты, как круг, шар, цилиндр на сцене.
Есть несколько шаблонных фигур, именуемых как елка и шашлык на шампуре.
